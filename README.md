# Antr

A simple queue management system build with JavaScript.

## Quickstart

To deploy the application you need [node](https://nodejs.org/en/) and [pnpm](https://pnpm.js.org/) installed :

```
pnpm install

pnpm run start
```

To build the application :

```
pnpm install

pnpm run build
```
