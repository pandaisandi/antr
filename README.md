<h1 align="center">Antr</h1>
<p align="center">
  A queue management system built for use in Bengkalis Primary Tax Service Office.
</p>

## Quickstart

To deploy the application you need node.js and npm installed :

```
npm install

npm start
```

<br>
To build the application :

```
npm install

npm run build
```
